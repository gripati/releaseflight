# Firebase App Distribution setup — guide screenshots

Drop your own screenshots here to replace the placeholders in the
"Deploy → Firebase → Connect" wizard. The wizard loads these files by exact name;
until a file exists, a dashed placeholder is shown instead.

| File | Step shown |
|------|------------|
| `step-1.png` | Register your app in Firebase (Add app) |
| `step-2.png` | Download the config file (GoogleService-Info.plist / google-services.json) |
| `step-3.png` | Enable App Distribution |
| `step-4.png` | Generate the service-account private key (Project settings → Service accounts) |
| `step-5.png` | Connect in Release Flight (drop files) |

PNG or JPG both work as long as the filename matches (e.g. `step-1.png`).
Recommended width ~1200px; they render full-width inside the wizard.
