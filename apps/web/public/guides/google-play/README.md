# Google Play setup — guide screenshots

Drop your own screenshots here to replace the placeholders in the
"Add credential → Google Play" wizard. The wizard loads these files by exact name;
until a file exists, a dashed placeholder is shown instead.

| File | Step shown |
|------|------------|
| `step-1.png` | Enable the Google Play Android Developer API (Cloud Console → API Library) |
| `step-2.png` | Create a service account |
| `step-3.png` | Create & download the JSON key (Keys tab) |
| `step-4.png` | Grant access in Play Console (Users and permissions) |
| `step-5.png` | Upload the JSON in Release Flight |

PNG or JPG both work as long as the filename matches (e.g. `step-1.png`).
Recommended width ~1200px; they render full-width inside the wizard.
