# Apple App Store Connect setup — guide screenshots

Drop your own screenshots here to replace the placeholders in the
"Add credential → Apple" wizard. The wizard loads these files by exact name;
until a file exists, a dashed placeholder is shown instead.

| File | Step shown |
|------|------------|
| `step-1.png` | Open the API Keys page (Users and Access → Integrations) |
| `step-2.png` | Generate an API key (name + access role) |
| `step-3.png` | Download the .p8 & copy the Key ID |
| `step-4.png` | Copy the Issuer ID |
| `step-5.png` | Enter the details in Release Flight |

PNG or JPG both work as long as the filename matches (e.g. `step-1.png`).
Recommended width ~1200px; they render full-width inside the wizard.
